## C# Format Number as Currency
MSDN: [String.Format Method](https://msdn.microsoft.com/en-us/library/system.string.format%28v=vs.110%29.aspx?f=255&MSPPError=-2147217396)
```csharp
Print(String.Format("{0:C}", 7.5));
// output: "$7.50"
```
